# Smart Air Quality Monitoring and Prediction System using IoT and Machine Learning

## 📌 Overview
The Smart Air Quality Monitoring and Prediction System is an IoT and Machine Learning based environmental monitoring solution designed to measure and analyze air pollution levels in real time. The system continuously collects environmental parameters such as gas concentration, temperature, and humidity using IoT sensors and transmits the data to the cloud for visualization and analysis.

The project integrates ESP32 microcontroller, MQ135 gas sensor, DHT11 sensor, ThingSpeak cloud platform, Flask web application, and Machine Learning algorithms to provide intelligent monitoring and prediction of air quality conditions. The system also includes an alert notification mechanism that warns users when pollution levels exceed safe thresholds.

This project was developed as a Final Year Capstone Project for Computer Science and Engineering with specialization in Artificial Intelligence and Machine Learning.

# 🧠 Machine Learning Algorithms Used

- Random Forest
- Logistic Regression
